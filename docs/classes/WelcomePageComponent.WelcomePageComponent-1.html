  • openUserLoginDialog(): void
    • Defined in welcome-page/welcome-page.component.ts:49

    If the user is already logged in, they should have a valid JWT saved in localStorage already. This function checks first for a JWT. If it's found, user is redirected to the movie card page.

    Otherwise, the login dialog should open.

    If the user has a JWT saved but it is invalid for some reason, the movie card page will find out when trying to get the movie data and the user will be logged out.
